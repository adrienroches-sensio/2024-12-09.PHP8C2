Formation PHP8C2 du 09 Décembre 2024 au 11 Décembre 2024
========================================================

# Pré-requis

* `php` >= 8.3

# Install

```shell
$ composer install
```

# Usage

## Fil rouge

```shell
$ ./run.php
```

## Design patterns

**Singleton**
```shell
$ ./patterns/singleton.php
```

**Proxy**
[Example](./patterns/proxy.php)

## Running tests

```shell
$ ./vendor/bin/phpunit --coverage-html ./coverage --testdox
```
